Introduction¶

A small notebook example on how to scrape data from pangaea

Imports¶

we start off by importing the libraries we need

  • request_html -> used for parsing html + javascript websites

  • …

from requests_html import AsyncHTMLSession
from bs4 import BeautifulSoup
import urllib

Define a Query / search¶

Here put the query you would look for on the website

query = 'Mass Accumulation Rate/Flux'

Run the code¶

# Store links
links = []
URL = 'https://www.pangaea.de/?q={query}&offset={offset}'
url_query = urllib.parse.quote_plus(query)
finished = False
offset=0
while not finished:
    print(offset)
    retry = 5
    while retry > 0:
        asession = AsyncHTMLSession()
        r = await asession.get(URL.format(query=url_query, offset=offset))
        await r.html.arender()
        resp= r.html.raw_html
        soup = BeautifulSoup(resp, 'html.parser')
        dataset_links = soup.find_all('a', class_='dataset-link')
        if len(dataset_links):
            retry = -1
        retry -= 1
    if not len(dataset_links):
        finished = True
    for anchor in dataset_links:
        link = anchor.get('href')
        link = "/".join(link.split('/')[-2:])
        links.append(link)
    offset += 10
